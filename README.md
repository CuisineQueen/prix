## Troubleshooting

### Common Issues

**Permission Denied**
```bash
# Run with sudo for full system access
sudo python3 main.py
```

**Port Already in Use**
```bash
# Find and kill Prix Security processes
ps aux | grep prix-security
sudo kill -9 <PID>

# Or use seamless launcher stop command
python3 seamless_launcher.py --stop
python3 seamless_launcher.py
```

**Missing Dependencies**
```bash
# Reinstall dependencies
pip3 install -r requirements.txt
```

**Database Locked**
```bash
# Stop Prix Security and restart
python3 seamless_launcher.py --stop
python3 seamless_launcher.py

# Or manually remove lock file
rm -f /tmp/prix-security.lock  # Linux
rm -f C:\temp\prix-security.lock  # Windows
rm -f ~/.prix/prix-security.lock  # macOS/Android/iOS
```

**IoT Device Not Detected**
```bash
# Check IoT status
python3 -c "from iot_integration import IoTManager; print(IoTManager().get_iot_status())"

# Restart IoT monitoring
python3 seamless_launcher.py --stop
python3 seamless_launcher.py

# Manually scan for devices
python3 -c "from iot_integration import IoTManager; IoTManager().detect_local_devices()"
```

### Log Analysis
```bash
# View real-time logs
tail -f /var/log/prix-security/prix.log  # Linux
tail -f ~/.prix/logs/prix.log  # macOS/Android/iOS
Get-Content -Wait "C:\ProgramData\PrixSecurity\logs\prix.log"  # Windows

# Search for errors
grep "ERROR" /var/log/prix-security/prix.log  # Linux
grep "ERROR" ~/.prix/logs/prix.log  # macOS/Android/iOS

# View threat history
sqlite3 prix_security.db "SELECT * FROM threats ORDER BY timestamp DESC LIMIT 10;"

# Monitor IoT device logs
tail -f /var/log/prix-security/iot.log  # Linux
```
